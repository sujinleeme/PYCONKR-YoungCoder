우리는 세가지 데이터 타입에 관하여 이미 알고 있습니다.

```
 “Hi!”                 string
    27                     integer
    15.238                 float

```

파이썬은 우리에게 `type()`함수를 사용한 타입에 대하여 이야기 할 수 있습니다.

```python
>>> type("Hi!")
<class 'str'>
```
여러분은 파이썬이 `int`형과 `float` 형을 출력하도록 할 수 있습니까?
