title: 파이썬과 말해봐요 - (에러) 오류를 고쳐보세요
---
{:.summary}
```
>>> "friend" + 5
Error
```

이 에러는 어떻게 고칠 수 있을까요?

연결은 작동하지 않습니다.
```
>>> "friend" + "5"
friend5
```
5를 문자열로 만들면 어떻게 될까요?

이 오류를 해결할 수 있는 또 다른 방법은 무엇일까요?

print 명령으로 새로운 것을 해봅시다.
```
>>> print("friend", 5)
friend 5
```

(print 명령과 함께 사용되는 쉼표와 두 오브젝트 사이에 공백을 추가하는 방법)
